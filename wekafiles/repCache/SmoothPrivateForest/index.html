SmoothPrivateForest: Smooth Private Forest for Differential Privacy

URL:https://github.com/zislam/SmoothPrivateForest/
Author:Michael Furner
Maintainer:Michael Furner <mfurner{[at]}csu.edu.au>

Implementation of Smooth Private Forest, a differentially private decision forest designed to minimize the number of queries required and the sensitivity of those queries.
For more information, see:
Fletcher, S., & Islam, M. Z. (2017). Differentially private random decision forests using smooth sensitivity. Expert Systems with Applications, 78, 16-31.
